README – Task 3: Sales Dashboard Design
Project Title

Sales Performance Dashboard Using Power BI

Objective

The objective of this project is to design an interactive sales dashboard that enables business stakeholders to monitor key performance indicators (KPIs), analyze sales trends, evaluate profitability, and make informed business decisions.

Tools Used
Power BI Desktop
Microsoft PowerPoint
Sales/Financial Dataset (CSV format)
Dataset Description

The dataset contains sales transaction records including:

Sales Amount
Profit
Quantity Sold
Product Information
Category Information
Payment Mode
Sales Type
Date (Year and Month)

The data was imported into Power BI and used to create interactive visualizations and KPI reports.

Dashboard Components
1. KPI Cards

The dashboard displays the following key performance indicators:

Total Sales
Total Profit
Profit Percentage
Total Quantity Sold
2. Time-Series Analysis

A monthly sales and profit trend chart was created to:

Track business performance over time
Identify growth patterns
Detect seasonal fluctuations
3. Sales by Product

A bar chart was used to display:

Best-selling products
Product performance comparison
4. Sales by Category

A category analysis chart was created to:

Compare category performance
Identify high-revenue categories
5. Sales by Type

Sales were analyzed according to:

Direct Sales
Online Sales
Wholesale Sales
6. Sales by Payment Mode

Payment methods were compared to determine:

Customer payment preferences
Revenue contribution by payment type
7. Interactive Filters (Slicers)

Users can interact with the dashboard using:

Year Filter
Month Filter
Sale Type Filter
Payment Mode Filter
Key Findings
Total Sales recorded: 366.62K
Total Profit achieved: 34.11K
Profit Margin: 10.26%
Personal Care and Household categories generated the highest sales.
Online and Cash payment methods contributed nearly equal revenue shares.
Sales remained relatively stable throughout the year.
Business Recommendations
Increase investment in high-performing product categories.
Promote top-selling products through targeted marketing campaigns.
Monitor monthly sales performance to identify growth opportunities.
Encourage the use of preferred payment methods to improve customer experience.
Use dashboard filters regularly for detailed performance analysis.
Conclusion

The Sales Dashboard provides a comprehensive view of business performance through interactive visualizations and KPI tracking. It supports data-driven decision-making by allowing stakeholders to analyze sales trends, profitability, product performance, and customer purchasing behavior efficiently.
